# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please avoid public disclosure.

Preferred options:

1. Open a private GitHub Security Advisory (if enabled).
2. If private reporting isn’t available, open a minimal public issue and request a private follow‑up.

Please include:

- A clear description of the issue
- Steps to reproduce
- Impact assessment (if known)
- Any suggested fixes or mitigations

We will acknowledge reports as soon as possible and coordinate a fix.
